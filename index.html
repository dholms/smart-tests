Smart Testing

This smart testing program is written entirely in plain javascript. I used acorn.js to parse the javascript code. I used acorn over Esprima primarily due to the smaller file size and quicker speed. Acorn also draws heavy inspiration from Esprima so much of the documentation worked both ways. The code for whitelist, blacklist, and structure checks is located in the smart_tests.js file. I then wired this code up to the CodeMirror in this webpage using code contained in application.js. I have sample data in the blacklist, whitelist, and structure variables right now which can be changed from application.js.

Simply type in the CodeMirror in order to see the issues update themselves.
